Every agent is one markdown file — YAML frontmatter (name, model, tools) plus its runbook — in .claude/agents/. Adding a specialist means adding a file there; nothing else registers it.

po.md is the only one with a standing learning cycle: once a P0 is fixed and verified, or a sprint ships, it looks back for durable patterns and either edits an agent file directly, adds a line to AGENTS.md → Postmortems / Learnings, or proposes a new agent for a real gap.
